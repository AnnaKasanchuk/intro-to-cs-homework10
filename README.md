# intro-to-cs-homework10

Назва проєкту
Обчислення математичного виразу за заданим значенням x (Variant 84)

Опис
Ця програма приймає від користувача дійсне число x, перевіряє, чи належить воно області визначення функції, та обчислює складний математичний вираз, що містить:
тригонометричні функції
логарифми
константи π та e
арктангенс
раціональні дроби
Якщо введене значення не належить області визначення, програма виводить повідомлення "result = undefined".
Також реалізовано:
Перевірку коректності введення
Обробку виключень
Інформаційний заголовок з автором та варіантом

Використані технології
C++17 / C++20
Стандартні бібліотеки:
<cmath> — математичні функції
<iostream> — введення/виведення
<stdexcept> — обробка помилок
<format> — форматоване виведення (C++20)
Область визначення
Програма працює для значень:
x > -6  
x ≠ -4  
x ≠ 10

Як запустити програму
1. Компіляція (g++ з підтримкою C++20)
g++ -std=c++20 -O2 main.cpp -o program
2. Запуск
./program
Після запуску програма попросить ввести значення x.
